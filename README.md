# 🔗 J GHOST LINK

## Advanced Social Engineering Assessment Tool

![Version](https://img.shields.io/badge/Version-2.0-green?style=for-the-badge)
![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Linux%20%7C%20macOS-lightgrey?style=for-the-badge)

A sophisticated social engineering assessment tool designed for penetration testing and security awareness training. J Ghost Link demonstrates how attackers can leverage deceptive techniques to capture sensitive information.

**This tool is intended for:**

**Usage requires explicit permission from targets. Unauthorized use is illegal and unethical. Developers assume no liability for misuse.**

## 🚀 Features

- **🌐 CloudFlare Tunneling** - Secure and anonymous URL forwarding
- **📍 GPS Location Tracking** - Precise geolocation capture with accuracy metrics
- **📷 Camera Access** - Optional camera capture capabilities
- **🌍 IP Address Logging** - Target IP address collection
- **🎭 Multiple Templates** - Customizable phishing scenarios:
  - Event Wishes
  - Live YouTube TV
  - Online Meetings
- **🖥️ Cross-Platform** - Compatible with Windows, Linux, and macOS
- **🎨 GUI Interface** - User-friendly terminal-based interface

## 📋 Prerequisites

- PHP 7.0 or higher
- Internet connection
- CloudFlare account (for tunneling)
- Appropriate legal authorization

## 🛠️ Installation

```bash
git clone https://github.com/jprojectplatform/J-Ghost-Link.git
cd J-Ghost-Link
chmod +x jghostlink.sh
```

## 🎯 Usage

1. **Start the tool:**
```bash
./jghostlink.sh
```

2. **Select tunneling option:**
   - Choose CloudFlare Tunnel (recommended)

3. **Choose template:**
   - Event Wishes
   - Live YouTube TV  
   - Online Meeting

4. **Customize the bait** (if applicable)

5. **Share the generated link** with authorized targets

6. **Monitor results** in real-time

## 📊 Captured Information

- Camera Images (if granted access)
